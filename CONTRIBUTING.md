# Contributing to Envoys8 Knowledge Base

感谢你的关注！以下是贡献指南。

## 如何贡献

### 提交 Issue

- 发现错误或有建议？欢迎提交 Issue
- 请尽量描述清楚问题和上下文

### 提交 Pull Request

1. Fork 本仓库
2. 创建你的分支 (`git checkout -b feature/AmazingFeature`)
3. 提交更改 (`git commit -m 'Add some AmazingFeature'`)
4. 推送到分支 (`git push origin feature/AmazingFeature`)
5. 打开一个 Pull Request

## 文档规范

- 使用 Markdown 格式
- 文件名使用小写，单词间用连字符分隔
- 在文件开头添加简短的标题和描述

## 行为准则

请遵守 [CODE_OF_CONDUCT.md](CODE_OF_CONDUCT.md)。
