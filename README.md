# Envoys8 知识库 👋

> 🧩 **个人知识管理与经验记录中心** — 一个不断积累的学习笔记、技术博客和经验总结仓库。

## ✨ 这是什么？

这个仓库是 **envoys8** 的个人知识库，用来：

- 📝 **写博客** — 技术文章、学习心得、项目总结
- 📚 **记笔记** — 读书笔记、教程整理、知识归档
- 🔧 **存经验** — 踩坑记录、解决方案、最佳实践
- 🌱 **做复盘** — 定期回顾、成长日志、目标追踪

## 📂 仓库结构

```
envoys8.github.io/
├── README.md              # 本文件

├── notes/                 # 学习笔记
│   ├── reading/           # 读书笔记
│   ├── tutorials/         # 教程整理
│   └── snippets/          # 代码片段
├── experience/            # 经验记录
│   ├── bug-fixes/         # Bug 修复记录
│   ├── solutions/         # 解决方案
│   └── best-practices/    # 最佳实践
├── docs/                  # 项目文档
│   └── insights/          # 每日/定期洞察

    └── workflows/         # 自动化工作流
```
